🎮 Modern Tic Tac Toe Game

A modern and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project features a stylish user interface, turn-based gameplay, winner detection, and a reset system for a smooth gaming experience.

🚀 Features
Two-player gameplay (X vs O)
Real-time turn indicator
Automatic winner detection
Draw game handling
Reset game functionality
Modern and responsive design
Smooth user interaction and animations
🛠 Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
📖 Project Overview

This project recreates the classic Tic Tac Toe game with a modern design and interactive gameplay. Players take turns placing their marks on the board, and the game automatically checks for winning combinations or a draw after every move.
